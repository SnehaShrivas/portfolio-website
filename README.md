# React Portfolio Website

A modern and responsive portfolio website built with React.js and Tailwind CSS. This project showcases my frontend development skills through a clean UI, smooth animations, and reusable React components.

## Live Demo 
portfolio-website-sneha-shrivas.vercel.app

## Technologies Used
- React.js
- Tailwind CSS
- JavaScript (ES6+)
- Vite
- Vercel

## Features
- Responsive layout for mobile, tablet, and desktop
- Smooth scrolling and interactive animations
- Reusable React components
- Modern and minimal UI
- Project showcase section
- Skills section
- Contact section
- Easy to customize

## Project Structure
```
src/
├── components/
├── assets/
├── pages/
├── App.jsx
└── main.jsx
```

## Installation

```bash
git clone https://github.com/SnehaShrivas/portfolio-website.git
cd portfolio
npm install
npm run dev
```

## Deployment

The project is deployed using Vercel.
